# .compress_and_get_size

Compresses an image and returns the compressed file size.

## Usage

``` r
.compress_and_get_size(img, flname, algorithm, use_existing_bmp)
```

## Arguments

- img:

  An array or matrix of numeric values or integer values.

- flname:

  Character string (file name without extension).

- algorithm:

  Character string (which compression algorithm to use).

- use_existing_bmp:

  Logical (use existing bmp file).

## Value

a numeric value (compressed file size)
